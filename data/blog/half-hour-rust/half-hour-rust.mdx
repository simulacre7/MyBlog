프로그래밍 언어에 익숙해지기 위해서는 그 언어로 작성된 코드를 많이 읽어야만 한다.
이 글에서 최대한 많은 Rust 코드 조각들을 다루고자 한다.
독자들은 Rust의 수 많은 키워드와 기호가 각각 어떤 의미를 갖는지 알게 될 것이고,
온라인에서 마주치는 대부분의 Rust 코드를 이해할 수 있게 될 것이다.

let은 변수를 선언한다:

```rust
let x; // declare "x"
x = 42; // assign 42 to "x"
```

한 줄로 줄여쓸 수도 있다:

```rust
let x = 42;
```
